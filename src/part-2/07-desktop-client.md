# 桌面客户端安装与登录

本章会覆盖主流 AI 工具桌面客户端的安装、登录和基础设置。

### 桌面客户端适合什么时候用

网页端足够完成大多数入门任务。等你开始频繁处理资料、截图或本地项目时，桌面客户端会更顺手：快捷键呼出、截图提问、文件拖拽、语音输入、本地项目协作，都更贴近日常工作。

### 安装原则

安装客户端前，先确认使用场景。如果只是偶尔问问题，网页端已经够用；如果你需要截图、拖文件、快捷键、语音或本地项目协作，再安装客户端就更合理。

确认需要之后，下一步是确认来源。只从官方渠道下载，避开来历不明的“增强版”“破解版”“镜像版”。客户端会接触更多本地文件和系统权限，来源可信很重要。

安装后先用同一个账号登录，完成一次普通对话；然后看它请求了哪些权限，比如文件访问、麦克风、屏幕录制、辅助功能、通知、剪贴板。需要哪个功能，再开哪个权限。

### 第一次打开后做什么

第一次打开后，先找到新建对话和设置入口，确认能正常聊天；再找文件上传或截图入口，用一份非敏感材料测试；最后再看快捷键、模型选择、语音等进阶功能。

### Windows 用户注意

如果中文显示异常，可能与系统语言、软件编码或终端编码有关。桌面软件一般不需要你手动改 PowerShell 编码，但如果你是在终端里运行相关工具，就可能遇到编码问题。

如果安装后打不开，先重启软件；如果仍然不行，再检查是否被安全软件拦截，或者更新到官方最新版。必要时先回到网页版继续练习。

### 检查点

```text
我确认客户端来自官方渠道。
我知道它申请了哪些权限。
我能完成一次普通对话。
我能上传一个非敏感文件或截图测试。
```
