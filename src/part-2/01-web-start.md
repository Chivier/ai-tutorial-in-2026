# 打开网页，开始对话

本章只负责把读者带到 `Claude`、`ChatGPT`、`Kimi` 和 `DeepSeek` 的网页端入口，并确认已经能看到对话输入框。

这一章只要完成一条路径：

先打开官方网页入口，再用平台支持的方式登录；只要能进入聊天页面、看到输入框，就可以进入下一章发出第一条消息。

建议优先从你最容易访问的平台开始。能用哪个就先用哪个，先把学习流程跑起来。

### 如何确认是官方入口

打开平台前，先确认入口来源。尽量从官方主页、官方应用商店、可信文档或你确认过的链接进入。

如果搜索结果里有广告、镜像站、代充站，先回到官方入口确认。涉及账号、验证码、支付信息或 `API Key` 时，入口可信比速度更重要。

你可以把“官方入口”理解成安全检查。只要涉及账号、付费、密钥，就值得多确认一次。

### 进入页面后看什么

不同平台界面不同。第一次进入时，先找到**输入框**即可。

输入框通常旁边会有新建对话、模型选择、上传文件、语音、历史记录、账号设置等入口。这些功能后面会慢慢用到，本章先不展开。

如果需要登录，就按页面提示完成邮箱、手机号、第三方账号或其他验证。

### 如果遇到问题

如果某个平台暂时打不开、无法验证或提示地区限制，可以先换另一个平台继续。

如果你担心点错页面，可以把当前网址、页面截图或错误提示发给 AI，让它帮你判断下一步。验证码和密码留在自己手里。

### 检查点

完成本章后，你应该能确认：

```text
我至少打开了一个 AI 工具的官方网页。
我已经登录或进入可使用状态。
我能看到聊天输入框。
我知道下一章要在这个输入框里发出第一条消息。
```
