# 发出第一条消息

本章会带读者完成第一轮对话，并说明第一条消息怎样写得轻松、有效。

### 第一条消息可以很简单

第一次打开 AI 时，可以先从一个容易回答的问题开始。第一条消息只需要启动对话，后面可以慢慢调整。

可以先发一个低压力的问题：

```text
你好，我正在学习如何使用 AI。请用简单的话告诉我，我可以让你帮我做哪些事。
```

也可以问一个本教程相关的问题：

```text
请用普通人能听懂的话解释一下：什么是 AI 大模型？
```

发送后先观察它的回答是否清楚，是否有你想继续问的地方。

### 试一次追问

追问时，先处理最明显的问题，一次调整一个方向。

如果回答太长，可以说：

```text
请再短一点，用 5 条要点说明。
```

如果回答太抽象，再让它举例：

```text
请给 3 个生活中的具体例子。
```

如果你没看懂，就让它降低难度：

```text
我还是没懂，请用初中生也能理解的方式解释。
```

AI 对话最基本的能力是：**你可以不断调整答案**，而不是一次问完就结束。

### 改变输出格式

当内容方向已经对了，再改输出格式。

```text
请把刚才的内容整理成表格，包含三列：用途、例子、注意事项。
```

或者：

```text
请把它改成一份 7 天入门练习计划。
```

格式要求越具体，结果越容易直接使用。

### 小练习

完成下面四步：

1. 问 AI 一个你真的关心的问题。
2. 让它把答案变短。
3. 让它举例。
4. 找出一个你不确定的事实，自己核查。

这个练习的重点是体验 **提问 -> 追问 -> 改格式 -> 核查** 的基本循环。
