# Web vs 客户端

本章会用具体场景说明什么时候用网页端，什么时候用桌面或手机客户端。

### 没有唯一正确入口

`Web`、桌面客户端、手机 `App` 都是 AI 工具的入口。选择入口时，看当前场景。

### 什么时候用 Web

如果你是新手，`Web` 端通常是最轻的起点。打开网页、登录账号、看到输入框，就可以开始练习提问、追问、总结和改写。

所以第一次学习时，先用 `Web`。等你发现自己经常要上传文件、处理截图、频繁打开同一个工具，再考虑客户端。

### 什么时候用桌面客户端

桌面客户端适合从“偶尔使用”进入“日常工作流”的阶段。你开始频繁处理截图、文件、本地项目，或者希望用快捷键随时呼出 AI，这时客户端会更顺手。

判断顺序是：先确认自己确实需要这些能力，再安装官方客户端，最后按需开启权限。

### 什么时候用手机 App

手机 `App` 适合现场问题。看到说明书、菜单、路牌、设备报错时，拍照比打字快；走路或通勤时，语音输入比长篇打字自然。

### 推荐组合

新手可以先这样搭配：

```text
Web 端：主力学习和普通任务
手机 App：拍照、语音、随手记录
桌面客户端：等高频使用后再安装
```

等你进入 `Agent` 和项目协作阶段，再把终端、`Git`、`Codex` 或 `Claude Code` 加进来。
